# MovApps
Create Movie Application using Kotlin 

## Features
1. Authentication
2. Onboaring
3. Splashscreen
4. Choose Seat
5. Checkout
6. Purchase Tickets
7. User Settings
8. Realtime database
9. Payment gateway (Incomming)
